# Insurance-Dashboard-Power-BI-Project
This project is an Insurance Dashboard created using Power BI to analyze insurance-related data through interactive charts, graphs, and reports. The dashboard helps users understand policies, claims, customer insights, and overall business performance.

## Dashboard Features
- Interactive Power BI dashboard  
- Insurance claims analysis  
- Policy performance tracking  
- Customer insights visualization  
- Dynamic charts and graphs  
- Filters and slicers for detailed analysis  
- Clean and professional design  

## Tools & Technologies Used
- **Power BI**
- Data Analysis
- Data Visualization

## File Included
-  Main Power BI dashboard file

## How to Use
1. Download this repository.
2. Open `Insurance Dashboard.pbix` in **Microsoft Power BI Desktop**.
3. Use filters and slicers to interact with the dashboard.
4. Explore insurance data insights.

## Project Purpose
This project is useful for:
- Students learning Power BI  
- Beginners practicing dashboard creation  
- Insurance data analysis projects  
- Learning interactive reporting techniques  

## Insights You Can Get
- Claim trends and status  
- Policy distribution analysis  
- Customer demographics insights  
- Revenue / premium performance  
- Overall business trends
